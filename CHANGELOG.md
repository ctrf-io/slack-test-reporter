### Added
- Threading support: Added `--thread-ts` (alias `--tt`) option to post messages as threaded replies
- Timestamp output: Added `--return-ts` (alias `--rt`) flag to output message timestamp for use in subsequent threaded replies
- Threading works with both OAuth token and webhook authentication methods
